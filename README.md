# Personal Expense Tracker

## Project Description
The Personal Expense Tracker is a web-based application that helps users manage their income and expenses efficiently. The system allows users to record transactions, categorize spending, analyze financial data through charts, and track their overall balance.

## Objectives
- Track income and expenses.
- Categorize transactions.
- Monitor financial balance.
- Visualize spending patterns.
- Export transaction records as CSV.
- Provide responsive and modern UI.

## Technologies Used
- React
- TypeScript
- Tailwind CSS
- Recharts
- Local Storage API
- Vite

## Applications
- Personal Finance Management
- Student Budget Tracking
- Household Expense Monitoring
- Monthly Financial Planning
